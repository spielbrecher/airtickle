# airtickle
Ассистент для написания научных статей
Телеграм-бот @airtickle_bot

Формулирует гипотезу исследования на основании введенного названия и краткого описания (примерного) статьи. 
Генерирует ключевые слова и переводит на английский язык.
Ищет полезные статьи по теме исследования и выводит ссылки для их скачивания.

Версия в блокноте GigaChat.ipynb может также сохранять список литературы в Excel-формат и скачивать PDF файлы из архива статей
